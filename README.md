# Asteroids (Python Pygame)

Dette er en enkel implementasjon av det klassiske arkadespillet Asteroids, bygget med Python og Pygame-biblioteket. Målet er å styre romskipet ditt, skyte asteroider og unngå kollisjoner.

## Installering

Følg disse trinnene for å sette opp prosjektet lokalt:

1.  **Klon repositoryet:**
    ```bash
    git clone <repository_url>
    cd asteroids
    ```
    (Merk: Erstatt `<repository_url>` med den faktiske URL-en til dette repositoryet, hvis dette er et klonet prosjekt.)

2.  **Installer avhengigheter:**
    Prosjektet krever `pygame`. Du kan installere det ved hjelp av `pip`:
    ```bash
    pip install -r requirements.txt
    ```

## Slik starter du spillet

Etter at avhengighetene er installert, kan du starte spillet ved å kjøre `main.py`-filen:

```bash
python main.py
```

## Kontroller

*   **W / Opp-pil:** Akselerer romskipet fremover
*   **S / Ned-pil:** Brems romskipet bakover
*   **A / Venstre-pil:** Roter romskipet mot venstre
*   **D / Høyre-pil:** Roter romskipet mot høyre
*   **MELLOMROM:** Skyt

Ha det gøy!
